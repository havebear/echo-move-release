# Echo Move 0.0.1

首个公开版本。

Echo Move 是一个面向个人运动数据的桌面应用，支持导入、整理、修复、可视化与分析 `FIT` / `GPX` / `TCX` 文件。应用采用 local-first 设计，活动数据默认保存在本地，不依赖账号体系或云端同步。

## 本次发布亮点

- 支持 `FIT`、`GPX`、`TCX` 运动文件导入、导出与批量处理
- 提供活动列表与轨迹墙两种浏览方式
- 提供地图轨迹渲染、筛选、最近活动高亮、截图与海报导出
- 提供生涯总览、趋势统计、年度汇总和热力图视图
- 内置格式转换、数据修复与轨迹修复工具
- AI Lab 已支持归纳整理统计数据，并可根据运动数据生成图表

## 主要功能

### 活动管理

- 支持文件选择、拖拽和批量导入
- 支持按关键词、运动类型、时间范围等条件快速筛选
- 支持批量导出
- 兼容不同坐标系统
- 活动数据默认仅保存在本地

### 地图

- 基于 MapLibre GL 的轨迹渲染
- 支持轨迹颜色、线宽、透明度与底图样式切换
- 支持独立地图筛选、图例、统计卡片和最近活动高亮
- 支持隐私模式、截图导出和海报导出

### 统计与 AI

- 提供生涯总览、趋势统计、年度汇总和热力图
- AI Lab 已支持对统计数据做归纳整理
- AI Lab 已支持根据运动数据直接生成图表

### 工具与设置

- 提供格式转换与数据修复工具
- 支持多用户本地档案
- 支持主题、语言和存储管理

## 安装包

- macOS (Apple Silicon): `Echo Move-0.0.1-arm64.dmg`
- macOS (Intel): `Echo Move-0.0.1-x64.dmg`
- Windows (64-bit): `Echo Move-0.0.1-setup.exe`
- Linux (64-bit): `Echo Move-0.0.1-x64.AppImage`
- Linux (64-bit): `Echo Move-0.0.1-x64.deb`

## 使用说明

- 首次使用前可直接导入本地 `FIT` / `GPX` / `TCX` 文件
- AI Lab 需要先在应用设置中配置可用模型提供商
- 当前版本以本地数据管理为主，未提供云同步

## 已知说明

- `0.0.1` 为首个公开版本，后续仍会持续打磨交互和细节体验
- 部分 AI 能力依赖外部模型配置与可用额度
